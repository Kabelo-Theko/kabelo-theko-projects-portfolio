# kabelo-theko-projects-portfolio

A single-page portfolio for Kabelo Theko, IT Support Technician. It introduces
three small, working tools that cover the first-line support loop end to end,
each with a screenshot, a short description, and links to the live demo and the
source.

**The projects**

- [ticket-triage](https://github.com/Kabelo-Theko/ticket-triage) — sorts raw
  support messages into structured, prioritised tickets. Live:
  https://ticket-triage-nu.vercel.app/
- [net-doctor](https://github.com/Kabelo-Theko/net-doctor) — a layered network
  troubleshooting decision tree. Live: https://net-doctor-nu.vercel.app/
- [onboard-kit](https://github.com/Kabelo-Theko/onboard-kit) — generates a full
  IT onboarding pack from a new starter's details. Live:
  https://onboard-kit-eta.vercel.app/

## Running it

It is a single self-contained `index.html` (styles and images inlined), so there
is nothing to build. Open the file, or host it anywhere static.

- **Vercel:** import this repo at vercel.com/new and deploy; `vercel.json` keeps
  it a plain static site.
- **GitHub Pages:** Settings > Pages > Deploy from a branch > `main` / `/ (root)`.
